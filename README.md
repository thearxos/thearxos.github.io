## ğŸ”¥ CPU Governor â€” Performance System

ARXOS includes the **custom CPU Governor** pre-installed and pre-configured. This C-based performance controller is integrated into the system for maximum performance.

### Source Code

- **Written in C** â€” Maximum performance, minimal overhead
- **MIT License** â€” Open source

### Features
- âš¡ **Maximum Performance Mode** â€” Forces all CPU cores to max frequency
- ğŸ”§ **Intel & AMD Support** â€” Auto-detects CPU vendor and applies optimal settings
- ğŸš€ **Turbo Boost Control** â€” Enables/disables CPU turbo intelligently
- ğŸ”„ **Systemd Service** â€” Can be enabled to auto-apply on boot
- ğŸ“Š **Real-time Monitoring** â€” Display current frequencies and governors

### Usage

### Download

ARXOS ISO is approximately **~30GB** due to all 2840+ BlackArch tools being pre-installed.

**Download**: [thearxos.github.io/portal](https://thearxos.github.io/portal.html)

```bash

sudo dd if=arxos-2026.2-x86_64.iso of=/dev/sdX bs=4M status=progress

# Verify SHA256 checksum
sha256sum arxos-2026.2-x86_64.iso
```

### Graphical Installation with Calamares

ARXOS uses **source-built Calamares 3.4.1 (Qt6)** for installation.

1. Boot the ISO and launch the installer from the desktop or boot menu
2. Select your language and timezone
3. Configure disk partitioning â€” automatic or manual
4. Optionally enable full disk encryption (LUKS)
5. Create your user account and set hostname
6. GRUB bootloader will be configured automatically
7. Reboot and enjoy

### Recommended Partition Scheme

```
/boot     â€” 512MB   (ext4, unencrypted)
/         â€” 80GB+   (ext4 or btrfs)
/home     â€” Rest    (ext4 or btrfs)
swap      â€” 8-16GB  (optional)
```

> ARXOS requires more disk space than typical distributions because all BlackArch tools are pre-installed. Plan for at least 80GB on root.

### First Boot Checklist

